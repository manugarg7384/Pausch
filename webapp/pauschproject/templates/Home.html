Rules

Click on blocks to toggle lights on or off. Clicking a block has two results:

1. that block changes states
2. the blocks to the immediate left and right also change states

To win, continue toggling blocks until all of the bridge lights have been turned on.

About

Lights On! is an interactive game by Hunter Modisett, Leslie Bloomfield, Youzi Wu, Felipe Vargas, Jeffrey Zhang, Manu Garg, and Kevin Huang. It was designed and built for the Spring 2015 mini-course Interaction & Expression Using the Pausch Bridge Lighting at Carnegie Mellon University.

It is played via a web-based interface that controls the visual output of the bridge LEDs in real-time. The web app has 22 colored blocks that correspond to 22 panels of colored light on the bridge.

The game is designed to be played collaboratively, so that all user inputs impact what is being toggled and players must work together to win.
